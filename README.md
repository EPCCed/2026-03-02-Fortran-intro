# ARCHER2 Online Course: Introduction to Fortran 2 - 4 March 2026

This repository contains the material for use in the Introduction to Fortran course to be delivered as an ARCHER2 Online Course from 2 to 4 March 2026. Pushes to the `gh-pages` branch will be reflected in the pages available at [https://epcced.github.io/2025-05-19-MO-Fortran-intro/](https://epcced.github.io/2025-05-19-MO-Fortran-intro/).

The material can be built locally with `make serve` or `make docker-serve`.
